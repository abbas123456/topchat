About topchat

Topchat lets you create and customise chat rooms that you can add to your own website for free.

How it works

All you need to is register with topch.at and you will then be able to create as many chat rooms as you want. When you create a chat room you will be given some html code that you can copy and paste onto your website and that's all you need to do. The chat room is hosted on topch.at so you don't have to install anything on your server or anything fiddly like that.
